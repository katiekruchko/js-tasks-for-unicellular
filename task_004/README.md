## Задание
Дан массив чисел. Найти минимальную разницу между двумя соседними элементами массива

## Аргументы функции:
массив целых чисел

## Результат:
целове число, минимальная разница между двумя соседними элементами

## Примеры:
```
(9, 5, 7, 10, 2)
2
```

#### Пояснение
разница между элементами 5 и 7 равна 2 (|5 - 7| = 2)

```
([22, 25, 12, 100, 33])
3
```
